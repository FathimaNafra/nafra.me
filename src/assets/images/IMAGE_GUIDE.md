# Image Guide for Portfolio

This guide helps you prepare and add images to your portfolio.

## 📸 Required Images

### 1. Profile Image (`profile.jpg`)
- **Location**: `assets/images/profile.jpg`
- **Recommended Size**: 400x400px (square)
- **Format**: JPG or PNG
- **Tips**: 
  - Use a professional headshot
  - Plain or minimal background
  - Good lighting
  - Smile and look approachable

### 2. About Section Image (`about.jpg`)
- **Location**: `assets/images/about.jpg`
- **Recommended Size**: 600x800px
- **Format**: JPG or PNG
- **Tips**:
  - Can be another professional photo
  - Or a photo of you working/at computer
  - Should complement your profile image

### 3. Project Images (`project1.jpg` to `project6.jpg`)
- **Location**: `assets/images/project1.jpg`, etc.
- **Recommended Size**: 1200x800px (landscape)
- **Format**: JPG or PNG
- **Tips**:
  - Screenshot of your project
  - Mockup of the interface
  - High-quality and clear
  - Show the main features

## 🎨 Where to Get Images

### For Profile/About Photos:
1. **Professional Photographer**: Best option for quality
2. **DIY with Phone**: Use good natural lighting, plain background
3. **AI Profile Generators**: Services like ProfilePicture.ai (use responsibly)

### For Project Screenshots:
1. **Your Own Projects**: Take screenshots or use screen recording
2. **Mockup Tools**:
   - [Figma](https://figma.com) - Design mockups
   - [Mockup World](https://www.mockupworld.co/) - Free templates
   - [Canva](https://canva.com) - Easy design tool

### Placeholder Images (Temporary):
If you don't have images ready:
- [Unsplash](https://unsplash.com) - Free high-quality photos
- [Pexels](https://pexels.com) - Free stock photos
- [Lorem Picsum](https://picsum.photos/) - Placeholder image service

## 🛠️ Image Optimization

Before adding images to your portfolio, optimize them:

### Online Tools:
1. **[TinyPNG](https://tinypng.com/)** - Compress PNG/JPG (Recommended)
2. **[Squoosh](https://squoosh.app/)** - Advanced compression
3. **[ImageOptim](https://imageoptim.com/)** - Mac app

### Why Optimize?
- ✅ Faster page loading
- ✅ Better user experience
- ✅ Improved SEO
- ✅ Less bandwidth usage

### Target Sizes:
- Profile: < 200KB
- About Image: < 300KB
- Project Images: < 500KB each

## 📝 Image Naming Convention

Use clear, descriptive names:
- ✅ `profile.jpg`
- ✅ `about.jpg`
- ✅ `project-ecommerce.jpg`
- ✅ `project-task-manager.jpg`
- ❌ `IMG_1234.jpg`
- ❌ `Screenshot 2024-01-01.png`

## 🎯 Quick Setup Checklist

- [ ] Take or find a professional profile photo
- [ ] Create or find an about section image
- [ ] Capture screenshots of your projects
- [ ] Compress all images
- [ ] Rename images appropriately
- [ ] Place images in `assets/images/` folder
- [ ] Update image references in code if needed
- [ ] Test portfolio to ensure all images load

## 💡 Pro Tips

1. **Consistency**: Use similar style/tone for all images
2. **Quality**: Better to have fewer high-quality images than many low-quality ones
3. **Alt Text**: Always provide descriptive alt text for accessibility
4. **Lazy Loading**: Already implemented in the portfolio for better performance
5. **Backup**: Keep original high-res versions of your images

## 🚀 Advanced: Creating Project Mockups

### Using Figma (Free):
1. Create a new file
2. Use device frames (iPhone, MacBook, etc.)
3. Insert your project screenshots
4. Export as PNG at 2x resolution
5. Compress before uploading

### Using Canva (Free):
1. Search for "website mockup" templates
2. Replace placeholder with your screenshot
3. Download as PNG
4. Compress before uploading

## 📱 Responsive Image Tips

The portfolio automatically handles responsive images, but here are tips:

1. **High Resolution**: Provide 2x resolution for retina displays
2. **Aspect Ratios**:
   - Profile: 1:1 (square)
   - Projects: 3:2 (landscape)
   - About: 3:4 (portrait)

## ❓ Need Help?

If you don't have projects yet or images:

### Temporary Solutions:
1. Use placeholder images from Unsplash
2. Create mockups of planned projects
3. Screenshot class assignments (if they look professional)
4. Use personal projects, even if small

### Long-term:
1. Build 2-3 solid projects to showcase
2. Take professional photos when possible
3. Create a consistent visual brand
